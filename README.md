# ericjypark/homebrew-tap

Homebrew tap for [@ericjypark](https://github.com/ericjypark)'s macOS apps.

## Available casks

| Cask | App | Source |
|------|-----|--------|
| `codexisland` | [CodexIsland](https://github.com/ericjypark/codex-island) — AI usage limits in your notch | [`Casks/codexisland.rb`](Casks/codexisland.rb) |

## Install

```sh
brew install --cask ericjypark/tap/codexisland
```

The first invocation auto-taps this repo. Subsequent installs work with the
short form `brew install --cask codexisland` once the tap is registered.

## Updates

Each cask is auto-synced from its source repo's release workflow — version and
SHA-256 here always match the latest GitHub Release of the upstream app.
